# Direct Launch

## Linux
```bash
cd bc_thesis_tichavsky/WorldsFactory/publish/net7.0/linux-x64/
chmod +x WorldsFactory
./WorldsFactory
```

## Windows
```bash
cd bc_thesis_tichavsky/WorldsFactory/publish/net7.0/win-x64/
.\WorldsFactory.exe
```

# Build and Run

`.NET 7.0 SDK` required

```bash
cd bc_thesis_tichavsky/WorldsFactory/
dotnet restore
dotnet build
dotnet run
```
